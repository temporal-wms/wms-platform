====================================================================================

$ kubectl port-forward -n {{ .Values.global.namespace }} svc/order-service 8001:8001
$ curl http://localhost:8001/health

# Waving Service
$ kubectl port-forward -n {{ .Values.global.namespace }} svc/waving-service 8002:8002

====================================================================================
TROUBLESHOOTING:
====================================================================================

If pods are not starting:
1. Check pod events: kubectl describe pod <pod-name> -n {{ .Values.global.namespace }}
2. Check logs: kubectl logs <pod-name> -n {{ .Values.global.namespace }}
3. Verify secrets exist: kubectl get secret mongodb-credentials -n {{ .Values.global.namespace }}
4. Verify infrastructure is running:
   - MongoDB: kubectl get pods -l app.kubernetes.io/name=mongodb -n {{ .Values.global.namespace }}
   - Kafka: kubectl get pods -l app.kubernetes.io/name=kafka -n {{ .Values.global.namespace }}
   - Temporal: kubectl get pods -l app.kubernetes.io/name=temporal -n {{ .Values.global.namespace }}

For more help, visit: https://github.com/wms-platform/wms-platform
